Acerca de Esta Aplicación

Esta aplicación web presenta dilemas éticos complejos en el campo de la medicina para fomentar la reflexión y el análisis crítico.

Cómo Funciona

Utilizamos la potencia de la API de Google Gemini para generar escenarios realistas y variados cada vez que solicitas un nuevo dilema. Cada escenario incluye:

  • Un caso que describe una situación éticamente desafiante.
  • Una pregunta clave que el usuario debe considerar.
  • Opciones múltiples que representan diferentes cursos de acción posibles.

Una vez que seleccionas una opción, la aplicación (nuevamente, con la ayuda de Google Gemini) genera un análisis. Este análisis no busca juzgar tu elección como "correcta" o "incorrecta", sino explorar los principios éticos que podrían estar en juego y las posibles consecuencias de la opción seleccionada.

Objetivo

El objetivo principal es servir como una herramienta educativa y de autoexploración para estudiantes, profesionales de la salud y cualquier persona interesada en la ética médica. Permite enfrentarse a situaciones difíciles en un entorno simulado, promoviendo la consideración de diferentes perspectivas éticas (como la autonomía, beneficencia, no maleficencia, justicia, utilitarismo, ética del cuidado, etc.).

Los dilemas se generan cubriendo una amplia gama de áreas temáticas dentro de la medicina para ofrecer una visión más completa de los desafíos éticos.

Tecnología

La aplicación está construida con HTML, CSS y JavaScript puro para el frontend. Las llamadas a la API de Google Gemini se gestionan de forma segura a través de Vercel Serverless Functions para proteger la clave API.

Nota: Los casos y análisis son generados por una IA y deben considerarse como ejercicios de reflexión, no como asesoramiento médico o ético definitivo.
